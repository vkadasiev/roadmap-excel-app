# Roadmap Excel App

Локальное мини-приложение для отрисовки roadmap из Excel.

Откройте `index.html` в браузере и загрузите файл `.xlsx`, `.xls` или `.csv`.

## Формат Excel

Минимальный вариант:

| Продукт | Название | Плановая дата релиза |
| --- | --- | --- |
| ЕЛКП | Виджеты на главной | 01.06.2026 |
| ОФД | Раздел Аналитика | 15.06.2026 |

Если в файле есть колонка `Плановая дата релиза`, roadmap строит карточки именно по ней.

Для полосы вместо одной даты можно использовать колонки `Начало` и `Конец`, если колонки `Плановая дата релиза` нет.

Если дата написана без года, год берется из поля `Год` в приложении.
Строки roadmap строятся по колонке `Продукт`. Старые файлы с колонкой `Группа` тоже поддерживаются.
Цвет задавать в Excel не нужно: приложение назначает цвета продуктам само. Для нового продукта используется голубой цвет.
